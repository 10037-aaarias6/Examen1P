DOM Animation

El DOM Animación se refiere a la capacidad de utilizar JavaScript para manipular y controlar elementos HTML y crear efectos y animaciones dinámicas en una página web. Permite cambiar propiedades como el tamaño, la posición, el color, la opacidad, entre otros, de elementos HTML en respuesta a eventos. Esto se logra utilizando métodos y propiedades del objeto `HTMLElement` y manipulando su estilo y atributos a través de JavaScript. El DOM Animación mejora la experiencia del usuario, agrega interactividad y dinamismo a una página web, y se puede lograr tanto con JavaScript puro como con bibliotecas y frameworks especializados.

Ejemplo


                                <html>
                                <style>
                                #container {
                                  width: 400px;
                                  height: 150px;
                                  position: relative;
                                  background: yellow;
                                }
                                #animate {
                                  width: 50px;
                                  height: 50px;
                                  position: absolute;
                                  background-color: red;
                                }
                                </style>
                                <body>

                                <p><button onclick="myMove()">Click Me</button></p> 

                                <div id="container1">
                                  <div id="animate1"></div>
                                </div>

                                <script>
                                function myMove() {
                                  let id = null;
                                  const elem = document.getElementById("animate");   
                                  let posX = 0;
                                  let posY = 0;
                                  let directionX = 1;
                                  let directionY = 1;
                                  clearInterval(id);
                                  id = setInterval(frame, 10);
                                  
                                  function frame() {
                                    if (posX === 350 && posY === 350) {
                                      clearInterval(id);
                                    } else {
                                      posX += directionX;
                                      posY += directionY;
                                      
                                      if (posX === 350 || posX === 0) {
                                        directionX *= -1;
                                      }
                                      
                                      elem.style.left = posX + "px";
                                      
                                      if (posX % 100 === 0) {
                                        directionY *= -1;
                                      }
                                      
                                      elem.style.top = posY + "px";
                                    }
                                  }
                                }
                                </script>

                                </body>
                                </html>

DOM Events

Los eventos del DOM son acciones o sucesos que ocurren en elementos de una página web en respuesta a la interacción del usuario o a condiciones específicas. Permiten que la página web sea interactiva y responda a las acciones del usuario, como hacer clic en un elemento, mover el cursor sobre un elemento o enviar un formulario. Para manejar los eventos, se utilizan event listeners, que son funciones que se ejecutan cuando ocurre un evento específico en un elemento. Los eventos del DOM son fundamentales para crear páginas web dinámicas y mejorar la experiencia del usuario.


Ejemplos de eventos HTML:
1.Cuando un usuario hace clic con el mouse
2.Cuando una página web ha cargado
3.Cuando se ha cargado una imagen
4.Cuando el mouse se mueve sobre un elemento
5.Cuando se cambia un campo de entrada
6.Cuando se envía un formulario HTML
7.Cuando un usuario pulsa una tecla

Ejemplo


                     <html>
                     <head>
                       <style>
                        .box {
                          width: 100px;
                          height: 100px;
                          background-color: red;
                          margin-bottom: 10px;
                      }
                   </style>
               </head>
               <body>
                   <div class="box" id="box1"></div>
                   <div class="box" id="box2"></div>

                   <script>
    // Obtener referencia a los elementos del DOM
                    const box1 = document.getElementById('box1');
                    const box2 = document.getElementById('box2');

    // Agregar un event listener al primer cuadro (box1)
                    box1.addEventListener('click', function() {
                      box1.style.backgroundColor = 'blue';
                  });

    // Agregar un event listener al segundo cuadro (box2)
                    box2.addEventListener('mouseenter', function() {
                      box2.style.transform = 'scale(1.5)';
                  });

                    box2.addEventListener('mouseleave', function() {
                      box2.style.transform = 'scale(1)';
                  });
               </script>
           </body>
           <html>

DOM Event Listener

Un DOM Event Listener es una función o método utilizado en la programación web para detectar y responder a eventos en un documento HTML. Permite registrar una función que se ejecutará cuando ocurra un evento específico en un elemento del DOM, como hacer clic en un botón o mover el mouse. Es una herramienta fundamental para crear interactividad y añadir comportamiento dinámico a las páginas web.

DOM Navigation

El DOM Navigation se refiere al proceso de manipulación y exploración de los elementos y nodos dentro de una estructura de documento HTML o XML utilizando JavaScript. El DOM proporciona una representación en forma de árbol de la estructura de un documento, donde cada elemento, atributo y texto se considera un nodo. La navegación DOM implica acceder, buscar y modificar estos nodos mediante métodos y propiedades proporcionados por JavaScript. Esto permite a los desarrolladores interactuar con el contenido y la estructura del documento, agregar, eliminar o modificar elementos, cambiar estilos y clases, y responder a eventos. La navegación DOM es fundamental para la creación de interactividad y dinamismo en aplicaciones web.

DOM Nodes

Los DOM Nodes son los elementos individuales que componen la estructura de un documento HTML o XML. Cada elemento, atributo, texto y comentario en un documento se representa como un nodo en el árbol del DOM. Los nodos del DOM pueden tener relaciones jerárquicas entre sí, formando una estructura en forma de árbol, donde cada nodo tiene un padre, hijos y posiblemente hermanos. Los diferentes tipos de nodos incluyen elementos HTML, texto, comentarios, atributos y más. Los nodos del DOM son la base para acceder, manipular y modificar la estructura y contenido de un documento web utilizando JavaScript u otras tecnologías basadas en el DOM.

DOM Collections

Los DOM Collections se refieren a conjuntos de nodos que comparten ciertas características o propiedades y se agrupan en una estructura de datos proporcionada por el DOM. Las colecciones DOM son similares a las listas o arrays en otros lenguajes de programación, y permiten acceder y manipular conjuntos de nodos de manera conveniente. Algunos ejemplos comunes de colecciones DOM son HTMLCollection y NodeList. HTMLCollection es una colección de nodos que representan los elementos HTML en un documento, mientras que NodeList es una colección de nodos devuelta por métodos como querySelectorAll() o getElementsByTagName(). Estas colecciones facilitan la iteración y manipulación de múltiples nodos a la vez, lo que resulta útil al realizar operaciones como seleccionar elementos, recorrer listas de nodos o aplicar cambios en masa a través de bucles o métodos de manipulación de la colección.

DOM Node List

El DOM NodeList es una colección ordenada de nodos devuelta por métodos de consulta en el Modelo de Objeto de Documento (DOM). Aunque no es un array, se puede iterar utilizando bucles o métodos de iteración como forEach(). Proporciona acceso a nodos individuales mediante índices y facilita la manipulación de múltiples nodos simultáneamente. Es especialmente útil para realizar operaciones en lote, como modificar estilos, agregar o eliminar clases, acceder a atributos o realizar otras acciones en el contenido de un documento. Además, el NodeList permite conocer el número total de nodos en la lista y proporciona métodos y propiedades adicionales para la manipulación y navegación de los nodos.
